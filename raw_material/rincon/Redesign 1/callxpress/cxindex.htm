CALLXPRESS

CallXpress is a multi-application computer telephony server providing call processing, unified messaging, fax processing, and call center and customer service applications all within a Windows NT environment. It is a robust, scalable, system with a capacity of 128 voice ports in a single server. Along with a full compliment of standard call processing features, the CallXpress system sports a host of special application features. Many of these features are unique to CallXpress. Using a combination of the system's basic features and some of the special applications features, each customer can have a system tailored to meet their specific call processing and messaging needs. Most of these features are a standard part of the CallXpress product that require nothing more than to be programmed. One of the best justifications for the purchase of a voice and call processing system is the increase in employee productivity. The automated attendant feature, audio text applications, transaction processing and all of the other applications components can be used to offer increased call completion...more calls from customers get handled the first time. The automation of calls results in increased employee efficiency.
