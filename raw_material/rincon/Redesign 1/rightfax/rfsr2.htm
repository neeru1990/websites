RightFAX System Requirements

Minimum Server Requirements and Specifications:
Pentium 200, 64 MB RAM, 1GB hard drive, Microsoft Windows NT® Server v4.0 (SP3 or later), one or more fax boards from Brooktrout.

Desktop Operating Systems:
Windows 95, 98, NT4 and 2000

Email Systems Supported:
Microsoft Exchange and Mail, Lotus Notes, cc:Mail, Novell GroupWise, any SMTP/POP3 mail systems (requires optional RightFAX Email Gateway Module)

Address Books Support:
Outlook/Exchange, RightFAX address books, any ODBC address book, LDAP, MAPI

Inbound Routing Methods:
DDI, Voice assisted DTMF, DNIS, Manual, Line/Channel, CSID, ISDN, OCR (requires optional RightFAX Enhanced OCR module)
