RightFAX Enterprise Server 7.2 gives you:

Centralized management
Manage all your fax servers with point-and-click ease. View the status of every fax server on your network, start and stop fax services individually or globally, and determine server workload and configure least-cost routing rules.  

Intelligent Least-cost Routing & IP Fax Delivery
Delivers faxes via alternate RightFAX servers across your LAN, WAN, intranet or the Internet to ensure they are sent by the most economical means. You can also choose to send faxes via service providers over the Internet.

Intra-company Least-cost Routing
Delivers internal faxes via the RightFAX network of servers, bypassing the public-switched telephone network entirely, which provides significant savings for intra-company faxing.

Load Balancing
Maximizes system performance by directing outbound faxes to the server with the highest availability. 

Automatic Backup
Your RightFAX system is automatically backed up without any effort on your part, and without disruption of system operations.   

Paging and Short-message Service Support
Users can receive numeric pages with short status and notification messages for faxes received and sent. Administrators can receive fax server performance alerts via their pagers as well.

Cluster-aware Installation
RightFAX lets you make the most of your Microsoft Cluster Server configuration by automatically installing the appropriate components to the necessary cluster nodes.

International Character Set Support
A vast range of diacritical character sets is supported, allowing the RightFAX Enterprise Server solution to be deployed in your global offices in Europe, eastern Asia, South America and Africa.

NT Security Authentication
Administration of your system’s security is simplified by leveraging Windows NT account IDs and passwords; no need to maintain separate RightFAX passwords. 

Built-in E-mail Gateway
Supports all SMTP Internet mail systems, Lotus cc:Mail, Microsoft Mail and Novell GroupWise, allowing faxes to be sent and received from these popular e-mail applications. Benefit from advanced e-mail integration with Microsoft Exchange and Lotus Notes by purchasing add-on modules that maximize this functionality.

Automatic Fax Distribution
Faxes can be systematically distributed to groups of users for faster, more efficient processing. You can set distribution to “Linear” (round-robin distribution) or “Balanced” (distribution according to user processing rates).

Adjustable Bandwith Controls
Allows administrators to control fax traffic volume on a group-wide basis by restricting the number of pages or total number of faxes being sent simultaneously by a user within a specific time period.

RightFAX Reporter
A time-saving tool that helps you measure your fax server resources to make sure they are being used efficiently.
