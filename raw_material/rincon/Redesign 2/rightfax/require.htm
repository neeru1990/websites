RightFAX Fax Server Requirements

The computer designated as the RightFAX server must meet or exceed the following specifications:

  • Windows NT Server v4.0 or greater.
  • Pentium 90 MHz processor or equivalent. Pentium 200 MHz or faster is strongly recommended for optimal speed.
  • 64 MB of RAM. More RAM may be needed if the server will also run other services, such as SQL Server, SMS or Microsoft Exchange.
  • 540 MB hard drive or better. More disk space may be needed depending on a number of factors, including how long users are allowed to keep old faxes in the system. The RightFAX software itself requires 150 MB of disk space, but consider that each fax image (page) stored by RightFAX consumes an average of 35 KB. If disk storage space is a problem, users should be encouraged to delete their faxes after reading or printing them.
  • VGA adapter and monitor. This is required for Windows NT.
  • Network interface card. The RightFAX server must be connected to the local area network via a network interface card. RightFAX will work on most popular LAN topologies and is compatible with most manufacturers' network interface cards. Any networking protocol supported by Microsoft networking may be used:  NetBEUI, TCP/IP or SPX.
  • Microsoft Internet Explorer 5 (or higher) must be installed on the server. If it is not already installed on the RightFAX server machine, Internet Explorer will be installed as part of the RightFAX server installation.

RightFAX Production System Requirements

  • RightFAX Enterprise Server v7.0 or higher.
  • Windows NT Server v4.0 or greater, with Service Pack 5.0 or higher (installed on all servers).
  • Pentium 266 MHz processor or equivalent.
  • 256 MB RAM or greater.
  • 4 GB of uncompressed usable hard drive space in a fault-tolerant disk configuration.
  • Software or hardware RAID level 1 minimum.
  • Appropriate number and type (PCI or ISA) of full-length expansion slots.
  • Network interface card. The RightFAX server must be connected to the local area network via a network interface card. RightFAX will work on most popular LAN topologies and is compatible with most manufacturers' network interface cards. Any networking protocol supported by Microsoft networking, including NetBEUI, TCP/IP or SPX, may be used.
  • CD-ROM.
  • SVGA display card with monitor settings of at least 800 x 600 dpi resolution and 256 colors.
  • 33.6 Kbps or faster modem (required for production technical support).
  • Microsoft Internet Explorer 5 (or higher) must be installed on the server.  If Internet Explorer is not already installed on the RightFAX server machine, it will be installed as part of the RightFAX server installation.
  • pcANYWHERE 32 v8.0 (required for production technical support).
  • Local administrator rights to all RightFAX servers.

RightFAX Fax Board Requirements

RightFAX supports Brooktrout and Dialogic fax boards only. At least one fax board must be installed in the machine where the fax server software is running.

Top of page

RightFAX Phone Line Requirements

RightFAX can use several types of phone lines: loop-start (regular analog phone lines), Direct Inward Dial (DID), Integrated Services Digital Network (ISDN), and several types of digital lines.
