RightFAX versus the competition

As computer-based fax technology has matured, so has fax server software. And now, more than ever, it is important to realise that not all fax servers are created equal.

This guide is designed for the MIS, IT, CIO or Network Administrator tasked with evaluating fax server software. It comprises a test matrix for you to highlight key features of RightFAX which may or may not be found in other fax software.

The fundamentals
The following points summarise some of the most important factors that differentiate RightFAX from other fax server software:

RightFAX is truly an NT-based product - It was designed with the system administrator in mind. RightFAX uses the NT Services, the NT Registry and the NT Event Log to maximise operations.

RightFAX makes the best use of system resources - A 4-channel RightFAX system requires a 35 MB footprint. RightFAX also increases productivity and reduces overhead by launching threads instead of processes for tasks and initiating the same number of Services regardless of the number of channels in operation.

RightFAX is truly a client/server application - All fax information is stored in a central database on the server, not locally on the client, so your users always see the same fax information regardless of where or how they view their fax list.

RightFAX servers can be remotely monitored and dynamically reconfigured - RightFAX allows you to monitor servers from any Windows NT machine on the network without installing additional software and rarely requires you to down the server for the changes to take effect. Systems which rely on .INI files will require you to stop and start the system each time you make a change (e.g. add a user).

RightFAX offers Intelligent LCR - We supply administrators with features to make their job easier like intra-company routing, load balancing, redundancy, cascading routes, multiple wildcards and testing tools.

RightFAX allows clients to customise virtually every option available - or administrators can create group or global defaults and restrictions.

RightFAX provides security - RightFAX stores all fax data and user information in a secure, binary database. Users can never simply view sensitive passwords and other information in clear text.
