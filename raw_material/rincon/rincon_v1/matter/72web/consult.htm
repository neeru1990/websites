Installation and configuration services

Engineers perform installation and configuration of servers and modules in the RightFAX product line. These services can be performed either by an onsite engineer, or remotely via a dial-in or Internet connection, depending on customer requirements.

Upgrade and maintenance services

Upgrade and maintenance services allow customers to keep their RightFAX servers at the most current software levels as well as implement new features and configurations. Services can include the addition of new fax boards to the server, upgrading server software to the current version, addition of software modules to the server and supplemental onsite technical support. These services can be performed either by an onsite engineer, or remotely via a dial-in or Internet connection, depending on customer requirements.

Configuration review services

To ensure that our customers maximize their RightFAX solution on an ongoing basis, we provide configuration review services. Our engineers provide a complete review and analysis of the RightFAX configuration and provide recommendations for enhancements or modifications in the following areas:

  • System performance and tuning
  • System capacity analysis and planning
  • Deployment to users
  • Network and deployment planning
  • Possible integration with other applications

These services can only be performed by onsite engineering resources.

Advanced integration services

These services help customers integrate RightFAX with other business applications and leverage document delivery capabilities across the enterprise. Services include:

  • Advanced integration (multiple server, multiple location) with messaging applications such as Lotus Notes and Microsoft Exchange
  • Implementation of production components such as SAP, Oracle and Baan connectors and filters
  • Deployment of RightFAX in a Microsoft clustered server environment.

Most services can be performed either by an onsite engineer, or remotely via a dial-in or Internet connection. However, not all services are available via remote delivery.

Cover pages and forms services

This service provides customers with professional quality design and development of fax cover pages and forms. This service is provided on a remote basis only.
