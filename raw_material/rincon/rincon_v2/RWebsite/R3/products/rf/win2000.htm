Overview

To further our commitment to optimize our product line for the latest Microsoft Windows technology, RightFax has been tested and is supported on Windows 2000 products. RightFax is the first enterprise- and production-level fax server software to be certified for Windows 2000 Server and Advanced Server, and the RightFax client software is certified for Windows 2000 Professional. This page explains the difference between support and certification and defines the exact configurations supported or certified.

Find out why Windows 2000 certification is essential for your network and enterprise fax server and production e-document delivery systems: Patricia Seybold Group reports clearly greater value and reliability.

What is Supported?

RightFax has successfully tested all client and server software components with Microsoft Windows 2000. The RightFax 8.0 and 7.2 server is fully supported with Windows 2000. The RightFax 7.0 server software requires Service Pack 1A for version 7.0 in order to install to Windows 2000 machines.

While compatibility testing will continue with subsequent releases of both the RightFax server and Windows 2000, RightFax technical support will support Windows 2000 implementations today on versions defined in the configuration chart, below.

What is Certified?

RightFax products are certified to meet specific standards established by Microsoft. The Certified for Windows 2000 program is designed to let software developers take advantage of new technologies in Windows 2000 and build applications that are more manageable, more reliable, and reduce the cost of ownership for customers by lowering support and training costs. It also helps customers easily identify products that were designed specifically for the Microsoft Windows family of operating systems.

To receive the "Certified for Microsoft Windows 2000" logo, RightFax products completed and passed rigorous testing by an independent testing lab, Veritest. The RightFax clients listed below comply with stringent technical requirements regarding user interface, installation, data accessibility and manageability, component sharing and power management as outlined in Microsoft's Desktop Application Specification for Windows 2000. Additionally, RightFax server components are scheduled for certification in 2001. Please see the chart below for exact configurations.
