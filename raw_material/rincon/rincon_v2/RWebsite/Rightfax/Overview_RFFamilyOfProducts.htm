OVERVIEW: RIGHTFAX FAMILY OF PRODUCTS

The RightFax Universal Information Exchange line of products provides easy-to-use, easy-to-manage, enterprise-wide e-document delivery solutions that integrate with business applications to provide organizations with comprehensive information exchange capabilities. By combining fax, email, Internet and wireless technologies, Captaris provides a one-stop, full service solution that gives companies the freedom and control to tailor an information exchange environment to fit the growing and changing needs of their business.

RIGHTFAX ENTERPRISE FAX

Captaris RightFax enables businesses to increase employee productivity by automating fax delivery of user-created documents; reduce costs by eliminating manual faxing, paper and fax machines; and improve customer service by providing secure, confirmed and immediate document delivery.

RightFax Universal Information Exchange integrates with a variety of applications, including email, desktop, CRM, ERP, document management, imaging and scanning devices to make exchanging information electronically simple. RightFax Enterprise Server

  • Provides an enhanced notification feature utilizing a Certified Delivery process

Integration Module

  • Automates the delivery of business-critical documents such as invoices, purchase orders, statements and confirmations
  • Fax-enables any host or ERP application using Facsimile Command Language (FCL) commands and/or Embedded Codes
  • Enables Embedded Codes including host notification and add documents
  • Opens access to the RightFax XML Interface and RightFax Java API
  • Provides numerous methods of host notifications and delivery confirmations
  • Built to electronically deliver production-oriented, high-volume batches of documents directly from ERP legacy and mainframe applications without having to modify the data stream
  • Features document delivery capabilities such as forms overlay, flexible scheduling with multiple priority levels, faxing from multiple applications, and fax status notifications
  • Functionality is customized for organizations by Captaris Consulting Services, or an Authorized RightFax Production Partner

InternetLink

  • Delivers host documents via the Internet
  • Converts host documents to PDF, TIFF (Group 3 or 4), GIF or PCX images and attaches them to email messages or text

RightFaxConnector for SAP R/3

  • Provides a seamless interface from SAP R/3 to the RightFax Enterprise Server
  • Lets SAP R/3 users send and receive faxes directly from SAPoffice and all other SAP applications via SAPconnect
  • Unlimited user licenses

RightFax Connector for Panagon Image Services

  • Provides inbound fax conversion, indexing, routing and storage in Panagon Image Services
  • Facilitates high-volume outbound faxing from Panagon Image Services Repository
