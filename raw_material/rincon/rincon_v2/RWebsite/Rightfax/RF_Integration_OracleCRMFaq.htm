RightFax Integration: Oracle CRM FAQ
Q. What is the RightFax Connector for Oracle CRM?

A. The RightFax Connector for Oracle CRM is an optional module that provides seamless integration between Oracle CRM and RightFax, providing feature-rich RightFax faxing and e-document delivery capabilities to Oracle CRM users. The Connector interfaces with Oracle CRM at the foundation level providing centralized access to fax delivery functions for the more than 60 Oracle CRM modules. With RightFax, Oracle CRM users have immediate access to reliable and secure document delivery tools to speed response time to customers, improve worker productivity and streamline business communications.

Q. How does the connector work?

A. The RightFax Connector for Oracle CRM seamlessly integrates with Oracle CRM. RightFax Java Class Libraries provide the custom interface that allows for document sending directly from the Oracle CRM environment. Custom built for to handle unique Oracle document formats, the RightFax Java API reads aggregate HTML file contents and renders the contents as a fax image. Intuitive client interfaces make sending faxes as easy as sending email.

Q. What types of documents can be delivered from Oracle CRM with RightFax?

A. Oracle CRM, part of the Oracle e-Business Suite, offers integrated applications for marketing, sales, service and contracts that work over the web, phone and in the field. Documents can range from order confirmations, invoices and contracts to shipping information, fax broadcasts, product literature, as well as marketing and sales updates for customers, prospects and suppliers. With RightFax, users can send these business-critical documents directly from Oracle CRM applications, much as they would send or receive email messages. For outbound delivery, faxes can be automatically addressed using centralized Oracle contact information. Any document that is created, printed, manually faxed or mailed can be electronically delivered from RightFax saving considerable time and money.

Q. How does RightFax provide fax confirmations and notifications?

A. RightFax has comprehensive notification features so users can monitor their document delivery status. RightFax can send notifications to the fax originator -whether a host system, email address or other application. RightFax notifications can be customized to include a variety of fax information.

Q. How is the RightFax Connector for Oracle CRM installed and deployed?

A. Since RightFax integrates with Oracle CRM at foundation level, not on a module-by-module basis, deploying RightFax fax functionality across the Oracle enterprise is effortless.

Companies implementing multiple integrations with RightFax and Oracle applications may want to consider working with an Oracle administrator or consultant to assist with the integration. Captaris Implementation Services provides the knowledge and expertise for the RightFax server installation and the configuration of the RightFax Connector for Oracle CRM, as well as integrations with a host of other Oracle and enterprisewide business applications.

Q. How is RightFax administered?

A. World-leading RightFax is easy to use and manage, with an extensive array of administrative tools and resources. Intuitive interfaces for system management lets administrators manage RightFax servers easily, whether local or remote. And since RightFax is certified by Microsoft, companies can be assured higher reliability and lower cost of ownership.

Load balancing capabilities help maximize performance by preventing bottlenecks and delays in processing high volumes. Capabilities such as least-cost routing minimize telephone charges, letting a company see even faster return on investment.

Q. What are the requirements for installing the RightFax Connector for Oracle CRM?

A. Requirements include a RightFax 8.0 Enterprise Server with a RightFax 8.0 Connector for Oracle CRM installed. The RightFax Connector for Oracle CRM is a self-contained solution that installs all necessary components for a successful integration between Oracle CRM and RightFax. Once installed, the RightFax Java Class Libraries must be installed on the appropriate Oracle Server.

Q. How scalable is the RightFax Connector?

A. RightFax is highly scalable to meet expanding needs. Customers can start with a single server and as few as two fax channels, then expand to as many as 1,024 channels. As a modular solution, RightFax allows companies to start small and add more capabilities and capacity as their business grows.
