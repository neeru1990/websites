RIGHTFAX SYSTEM REQUIREMENTS
The computer designated as the RightFax server must meet or exceed the following specifications:
  • Windows NT Server v4.0 service pack 6 or later, Windows 2000 server. RightFax 8.5 is also compatible with the Windows XP platform
  • Pentium II 233 MHz processor.
  • 128 MB of RAM. More RAM may be needed if the server will also run other services such as SQL Server, SMS, or Microsoft Exchange.
  • 1 GB hard drive or larger. More disk space may be needed depending on a number of factors, such as how long users will be allowed to keep old faxes in the system. The RightFax software itself requires about 150 MB of disk space, but each fax image (a single page) stored by RightFax occupies an average of 35 KB.
  • Network interface card. The RightFax server must be connected to a local area network via a network interface card. RightFax will work on most LAN topologies and is compatible with most manufacturers network interface cards. Any networking protocol supported by Microsoft networking can be used, such as: NetBEUI, TCP/IP, or SPX.
  • Microsoft Internet Explorer 5.5.
  • VGA adapter and monitor. This is required for Windows NT.
  • CD-ROM drive.
