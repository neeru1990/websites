Built-in Email Gateway
Supports all SMTP Internet mail systems, Lotus cc:Mail, Microsoft Mail and Novell GroupWise, allowing faxes to be sent and received from these popular email applications. Benefit from advanced email integration with Microsoft Exchange and Lotus Notes by purchasing add-on modules that maximize this functionality.

Automatic Fax Distribution
Faxes can be systematically distributed to groups of users for faster, more efficient processing. You can set distribution to “Linear” (round-robin distribution) or “Balanced” (distribution according to user processing rates).

Adjustable Bandwidth Controls
Allows administrators to control fax traffic volume on a group-wide basis by restricting the number of pages or total number of faxes being sent simultaneously by a user within a specific time period.

International Character Set Support
A vast range of diacritical character sets is supported, allowing the RightFax Enterprise Server solution to be deployed in your global offices in Europe, eastern Asia, South America and Africa.

Fax Authorization System
Provides a method for management review of critical or sensitive faxes prior to their release. Faxes can be sent to a group’s administrator’s queue for approval. Reasons for approval or revision can be entered or stored within the fax file.

RightFax Reporter
A time-saving tool that helps you measure your fax server resources to make sure they are being used efficiently.

Windows 2000 Server Certification
RightFax 8.0 is certified on Windows 2000 Server and Advanced Server, including support for Windows Cluster Server. Certification provides application reliability, continuity in the user interface experience, and ease of installation and ongoing operation.

Conformance with Microsoft guidelines ensures the smooth operation of the RightFax server and reduces any risk of downtime that may be experienced with uncertified products. Certification on Windows 2000 Advanced Server takes this a step further by providing assurance that the RightFax server works in a Microsoft clustering environment.

Support for Microsoft Windows XP and Office XP
RightFax 8.0 is supported on the Windows XP and Office XP or "eXPerience" platform. This latest Windows environment has a completely new look and feel to the user interface.

Active Directory Synchronization
The Active Directory TM service is a central component of the Microsoft Windows ® 2000 operating system platform. Active Directory is the central authority that manages the identities and brokers the relationships between these distributed resources, enabling them to work together.

RightFax 8.0 leverages Active Directory and provides flexibility to the administrator in configuring the synchronization with RightFax. Administrators can now manage all user accounts, profiles, and related information from Windows. Using the RightFax snap-in for Microsoft Management Console (MMC), administrators can customize the synchronization to fit their organization's specific needs.

RightFax Enterprise Fax Manager Web Edition
For those occasions that system administrators want to perform maintenance or administer accounts from outside the office, RightFax 8.0 introduces Enterprise Fax Manager Web Edition (or "Web EFM"). Web EFM offers the ability to perform many of the administrative functions that can be accomplished from the EFM Windows client through a standard Web browser. RightFax system administrators can stop and start work servers, manage users and groups, monitor board services, and add billing codes from any PC that has a Web browser. The user interface for Web EFM is very similar to EFM in Windows. Web EFM is also ideal for enterprises that have implemented thin clients throughout their organization.

AutoReply
AutoReply lets the RightFax administrator create a customized reply that is automatically sent to the fax sender's CSID (caller subscriber identification) number when RightFax receives a fax. Using AutoReply in conjunction with automatic inbound routing, and RightFax becomes a part of the workflow management system. When the RightFax server receives an inbound fax, it will check to see if the inbound line has AutoReply enabled. If it does, an automatic reply, with a thumbnail of the first faxed page, will be sent back to the sender. The built-in flexibility of the AutoReply feature allows the RightFax administrator to choose which inbound numbers will send an automatic reply, how the reply fax will be formatted, and how the received fax will be handled (forwarding to a particular user, routing to a group, or printing to a specified printer).
