Captaris RightFax Add-On Modules & Connectors

RightFax add-on modules and connectors give you the flexibility and functionality to tailor your solution to meet your company's exact needs.

  • RightFax Gateway for Microsoft Exchange
  • RightFax Gateway for Lotus Notes
  • RightFax Web Client
  • RightFax PDF Module
  • RightFax OCR Converter
  • RightFax OCR Router
  • RightFax TeleConnect Module
  • RightFax Docs-on-Demand Module
  • RightFax Document Management Connector
  • RightFax SNMP Alerting Module
  • RightFax COM Module
  • RightFax SecureDocs
  • RightFax Connector for Panagon Image Services

Note: If you plan to purchase more than two modules, it may be more cost-effective to purchase the RightFax Enterprise Suite, a bundle of the RightFax Enterprise Server with the first eight RightFax modules listed above in one easy install.

These eight modules are included with the RightFax Enterprise Suite:

RightFax Gateway for Microsoft Exchange gives users one centralized place to send, receive and manage faxes and emails—through their Outlook email inbox—with seamless integration. RightFax 8.0 leverages Active Directory (AD) and provides robust synchronization capabilities. Fully customizable RightFax field mappings, by using the RightFax snap-in for Microsoft Management Console (MMC), allow administrators to perform all user maintenance from Windows.

RightFax Gateway for Lotus Notes makes sending and receiving fax documents as simple for users as using Notes mail. Advanced fax functionality is available to expand and simplify the process of sending faxes via Notes, all without modifications to the Notes mail template. Giving users tools to work within their familiar Notes environment reduces training time and streamlines communications processes. For the Notes administrator, native Notes installation and database synchronization between Lotus Notes/Domino and RightFax allows unified administration of both servers, saving time and resources.

Lotus Fax for Domino Customers: Learn about our special, limited-time offer for an easy migration to the RightFax for Lotus Notes solution.

RightFax Web Client lets users manage their fax mailboxes through all popular Web browsers, including Netscape Navigator, Microsoft Internet Explorer or any HTML 3.0 (and above) compliant browser. Users can send, receive, forward, route, delete and print faxes, attach files and perform other functions—all via the Web.

RightFax PDF Module allows you to take advantage of the universal PDF format by interpreting PDF documents and converting them to fax format perfectly every time. Because PDF files can be generated from virtually any application, there is no limit on the type of faxes that can be sent from RightFax with this module.

RightFax OCR Converter uses Optical Character Recognition technology to scan faxes for recognizable characters and convert them into editable document files that users can import into any application. Users can then work with the document, just as they would with a native application file. Advanced features like auto-flip and de-skew help to ensure accurate conversion every time

RightFax OCR Router uses Optical Character Recognition technology to scan the first page of an inbound fax for recognizable strings, match it to a corresponding routing table, and then deliver it to the appropriate mailbox. This module is designed to offer small divisions or departments (50 or fewer employees) a reliable inbound routing alternative that can be less expensive than purchasing individual DID numbers, easier to use than assigning DTMF extensions and less labor-intensive than manual routing.

RightFax TeleConnect Module allows access to faxes from any touch-tone phone, perfect for remote users or frequent travelers. Users can select specific faxes for retrieval or have all their documents sent to them. They can even forward their faxes to another network user - or to any fax number in the world.

RightFax Docs-on-Demand Module is a Web-based or phone retrieval system that lets you publish a document to your Web site or fax-on-demand system from a single library file. Users can then access these documents at any time.

Other RightFax add-on Modules, not included with the Enterprise Suite:

RightFax Document Management Connector seamlessly integrates RightFax with document management applications such as Hummingbird DOCS Open®   or the iManage™ infoRite™ client. The connection allows DOCS Open or infoRite users to easily fax directly from their client desktop, automatically fill in billing codes, add a history record after sending the fax, and import any existing faxes.

RightFax SNMP Alerting Module integrates with existing infrastructure to provide centralized, proactive problem detection and management. It issues SNMP alerts on behalf of the server, and allows services to be stopped and started from a management application. This is particularly useful in large, multi-server environments. Sample alerts include Low Disk Space, Send Queue Too Deep, Probable Line Failure and Server Improperly Restarted.

RightFax COM Module provides a window to the Component Object Model (COM) API for customizing RightFax to meet specialized faxing needs. With the COM Module, developers can deploy, integrate and implement systems more quickly at less cost. Additionally, the COM Module provides built-in security features, NT Authentication, and allows companies to leverage the full breadth of Microsoft products.

RightFax SecureDocs brings together the best of fax and email, offering companies a centralized, reliable, easy-to-use and manage e-document delivery solution that is reliable and secure. SecureDocs allows for email to be sent using digital encryption technology to secure a document's transmission from sender to receiver, and to protect against tampering. SecureDocs also provides an enhanced notifications feature utilizing a Certified Delivery process. By adding RightFax SecureDocs module to your RightFax Business or Enterprise Server, you gain the ability to send document securely, with the assurance that its intended recipient safely received the document.

RightFax Connector for Panagon Image Services provides enterprise-wide inbound and outbound faxing capabilities with the Panagon Image Services. The result is an efficient and cost-effective solution that integrates fax and document management technologies to allow companies to convert, index, route and store inbound fax documents in Panagon Image Services. In addition, RightFax integrates with Panagon Print module to allow users to conveniently fax outbound documents from their Panagon system. This seamlessly integrated solution increases efficiency, eliminates lost faxes and paper handling issues and improves overall document distribution by levering your technology investments.
