Overview

RightFax production systems provide critical-volume electronic delivery and receipt of mission-critical documents such as purchase orders, invoices and statements via fax, e-mail or the Internet. By implementing market-leading RightFax production systems to automate the exchange of mission-critical documents, you can:

  • Drastically reduce document delivery costs—by up to 90 percent.

  • Eliminate inefficient manual processes and improve reliability.

  • Improve response time to customers and suppliers.

  • Increase the speed and reliability of communications with trading partners and ensure just-in-time (JIT) ordering and delivery.

  • Streamline business-critical document processing and reduce days sales outstanding (DSO).

  • Receive notification that your mission-critical communications have reached their recipients.

  • Eliminate the need for multiple stand-alone fax machines.

With a RightFax production system, you benefit from a long list of features, including: immediate or scheduled delivery, broadcast fax, Intelligent Least-Cost Routing™, real-time delivery confirmation and tracking, forms overlay, and host notification. Our solutions provide you with the most comprehensive, enterprise-wide document delivery environment on the market today. RightFax production systems are based on Windows NT™ and Windows 2000 and adhere to industry standards for open systems in connections, protocols and topologies.

RightFax provides an enterprise-wide e-document delivery offering that automates your document delivery at any time from front-office to back-office electronic commerce applications. That’s why more than 80% of America’s Fortune 100 companies have replaced manually printing, mailing, or manually faxing mission-critical documents with our integrated RightFax electronic document delivery software offerings.

RightFax Integration Module

Combine your RightFax Enterprise Server or RightFax Enterprise Suite with the RightFax Integration Module and you have access to the most robust, reliable, production-level electronic document delivery system available today. This solution provides the tools to fax-enable any mainframe, host, legacy or ERP application, including Oracle, SAP, Baan, PeopleSoft, and more.

Output is directed to the RightFax Integration Module as if it were a printer. The RightFax Integration Module reformats the host output as unique documents, overlays the data onto specified electronic forms, and delivers it as finished documents to their recipients.

Support of XML and Java

For companies that need ultimate flexibility, the RightFax Integration Module provides the ability to extend the RightFax Server using the RightFax XML Interface™ and the RightFax Java API.

RightFax XML Interface

As a component of the RightFax Integration Module, the RightFax XML Interface provides the tools to convert XML to FCL. These tools allow organizations whose applications produce XML to integrate production-level faxing capabilities with these applications and enable queries, notifications and other actions.

RightFax Java API

Another component of the RightFax Integration Module, the RightFax Java API fax-enables applications that are written in Java. The RightFax Java API enables Java developers to rapidly integrate their Java applications using RightFax Java Class Libraries.

Production Add-Ons

Add-ons to the RightFax Integration Module provide additional functionality that allows you to provide integrated, automated document delivery from leading ERP applications and takes advantage of the power and savings of the Internet. 

RightFax Filter for Production tightly integrates with automated programs and host and server-based applications such as Oracle to provide production-level electronic document delivery. With RightFax Filter for Production, companies do not need to modify the data stream or embed FCL into their output. Raw data is sent from the application and FCL commands are then wrapped around the data to create the electronic document, which can then be delivered as a fax or e-mail.

RightFax Connector for Oracle E-Business Suite™ is an interface between the RightFax Integration Module and the Oracle Purchasing 11i application. Designed so that users can electronically deliver purchase orders, releases and change orders in the same way as they would if they were being sent to display or a network printer. Data that is typically printed from Oracle Purchasing 11i applications is simply sent to the RightFax Integration Module using the RightFax Connector for Oracle E-Business Suite. The proper form is accessed, the data is overlaid on the form, signatures and graphics are placed on the document, and the completed fax document is delivered via standard phone lines or the Internet. Companies that use other versions of Oracle applications can take advantage of RightFax production-level faxing with the RightFax Filter for Production.

RightFax InternetLink Module lets users send host documents as e-mail via the Internet. It converts documents to PDF, TIFF (Group 3 or 4), GIF, or PCX attachments and sends them as e-mail messages.

RightFax SecureDocs is used in conjunction with InternetLink to enable automated, secure and certified transmission of mission-critical documents using encryption technology via email. By providing this encryption capability, SecureDocs ensures confidentiality and protects documents against tampering. With SecureDocs, documents are posted on a Web site and can be accessed and opened by the intended recipient by using a password. When the recipient accesses the document, the sender is sent a notification that the certified document has been accessed.

RightFax Connector for Baan IV C3 provides critical-volume, unattended electronic delivery of mission-critical documents such as purchase orders and sales order acknowledgements directly from Baan IV C3 applications running on the Unix platform. Data that is typically printed from Baan IV C3 is sent to the RightFax Integration Module. The proper form is accessed, the data is overlaid on the form, signatures and graphics are placed on the document, and the completed fax document is delivered via standard phone lines or the Internet.

RightFax Connector for SAP R/3 provides a seamless interface from SAP R/3 to RightFax Enterprise Server, allowing SAP R/3 users to send and receive documents directly from SAPoffice™ and all other SAP applications via SAPconnect™.

The RightFax COM Module takes advantage of the Component Object Model (COM) application programming interface (API) and is a powerful add-on tool for customizing the RightFax product line. By utilizing the RightFax COM Module with RightFax 8.0*, companies can deploy and implement systems more quickly at less cost. In addition, COM provides built-in security features, NT Authentication and allows companies to leverage the full breadth of Microsoft products. When used in conjunction with the RightFax Integration Module or RightFax Connector for SAP, it provides commands and managerial tools to more rapidly fax-enable any host or ERP-based application using COM.

* The COM API does not run with RightFax 8.0 Business Server.

RightFax Production Suite

The RightFax Production Suite bundles the RightFax Enterprise Server, RightFax Integration Module, RightFax COM Module and RightFax Filter for Production in one comprehensive offering. It provides all the basic modules for fax-enabling legacy and host applications at an affordable price. Additional RightFax production modules can also be added to the RightFax Production Suite.

Included in the RightFax Production Suite are the following:

  • RightFax Integration Module
  • RightFax COM Module
  • RightFax Filter for Production
  • RightFax Enterprise Server
