TR1034™ Specifications

HARDWARE
PCI Platforms
  • Control CPU
    - PowerPC 8240@200MHz
  • DSPs: 6 TI 320C549@100MHz
  • Voice channels: Up to 30
  • Fax channels: Up to 30 (V.34)
  • Telephony Bus:
    - ECTF H.100; TDM capacity: 4,096 time slots
T1/E1 Interfaces & Signaling
  • T1 robbed bit: E&M (wink and immediate)
  • ISDN PRI: N.A. ETSI, Euro ISDN
  • R2 MFC
  • NFAS
FAX
  • ITU T.30; Group 3
  • V.17, V.29, V.27ter, V.21, V.34 modulation
  • Up to 33.6Kbps with auto fallback
  • Normal and fine resolution: 100x200, 200x200
  • Additional: 300x300, 200x400, 400x400
  • MH, MR, MMR compression
  • On-board image conversion
  • A4, A3 and B4 page sizes with scaling
  • ASCII conversion with headers
  • Image pass-through
    – Color fax T.42 (JPEG)
    – JBIG T.85 (B/W), T.43 (color)
  • Error Correction Mode (ECM)
  • Line error detection/repeat good line
VOICE PROCESSING
  • 64Kbps G.711 µ-law and A-law PCM
  • OKI ADPCM 6KHz & 8KHz sample rates
  • 11KHz 8/16 bit linear .WAV, 8KHz 16 Bit linear .WAV
  • DTMF/MF/Special Information Tone (SIT) Detection
  • Custom call progress tone generation
  • Playback volume control, pitch corrected speed control
  • Voice activated record
  • Auto record level control
  • Silence compression
  • Talk Off/Play Off rejection
  • Speech play monitoring
CALL PROGRESS AND CALL CONTROL
  • High call density with reduced CPU loading
  • International call progress and tone detection
  • Adaptive pattern recognition
  • Programmable call progress detection
  • Answering machine detection
  • Call progress tone generation
  • Positive voice control (human/recorded)
  • CED, CNG, v.21 modem detection
  • ANI/DNIS DTMF and MF detection
CERTIFICATION (INTEROPERABILITY) APPROVALS
US/Canada
  • Safety: UL 1950
  • Regulatory: FCC Part-68
  • Emissions: FCC Part 15 Class A
Europe
  • Safety: EN60950: 1992
  • Emissions: EN55022: 1994 Class B
  • Immunity: EN55024: 1998
  • ISDN: TBR04
  • 75 ohm: PD 7024: 1998

For updated certifications and approvals, please contact Brooktrout Technology.

Operating Systems
  • Microsoft Windows 2000 Server
  • Microsoft Windows NT 4.0 Server
  • Red Hat Linux 6.2, 7.1, 7.2, 7.3
  • Solaris SPARC 7 and 8 (32 and 64-bit drivers and API libraries for both versions, 7 and 8)
  • Solaris x86, 7
  • UnixWare 7.1.0

Other operating systems may be supported. Please contact Brooktrout Technology for details.

PHYSICAL AND ENVIRONMENTAL
  • Host System Interface: One full-size Universal PCI slot (PCI-SIG Specification 2.3 Compatible)
  • RJ48C T1/E1 connector (optional)
  • Operating temperature: 0°C–50°C
  • Humidity: 10%–95% non-condensing
POWER REQUIREMENTS

Base card: 11.5W

  • TR1034 +P8H-E1, 8 channel digital fax board with E1 interface, uPCI
  • TR1034 +P10H-E1, 10 channel digital fax board with E1 interface, uPCI
  • TR1034 +P16H-E1, 16 channel digital fax board with E1 interface, uPCI
