RightFax

SAP Integration

ENTERPRISE INTEGRATION

The ideal solution for companies with the need to exchange information from multiple applications across their enterprise including SAP, desktop, email, supply chain management, sales force automation, forms and imaging systems.

Integrates with: SAP R/2 and above, depending on the integration method used.

RightFax requirements: Includes the RightFax Enterprise server and all the RightFax optional module software. Boards, channel licenses, support and services sold separately. See server requirements for more specific information.

Features:

  • Integrates with multiple SAP applications via SAPconnect, SAPscript or SAP Smart Forms as well as other business applications including desktop, email, CRM, forms, imaging, host and legacy systems.
  • Delivers information in a variety of electronic formats for high volume transaction environments - unattended and automatically.
  • Includes RightFax Enterprise Suite and netowrk fax capabilities for additional functionality at a cost-effective price.
    • Opens access to the RightFax XML interface and RightFax Java API.
    • Provides numerous methods of host notification and delivery confirmations.
    • Delivers information via fax, email, over the Internet and as digitally encrypted PDF and/or certified documents.
    • Unlimited user licenses.
