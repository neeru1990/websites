RightFax

SAP Integration

OVERVIEW

RightFax for SAP solutions provides a flexible, easy-to-deploy, and easy-to-use solution for automating document delivery from SAP applications at an affordable price. It combines SAP certified integration and data recognition tools with leading-edge document delivery capabilities, including wireless delivery, and first-class Captaris support.

Integrates with: SAP R/3 3.1g and later.

RightFax requirements: Includes the RightFax Enterprise Server, RightFax Connector for SAP, RightFax Integration Module, Captaris wireless platform and premium support plan. Boards, channel licenses sold separately. See server requirements for more specific information.

Highlights:

  • Integrates with SAP application via SAPconnect, SAPscript or SAP Smart Forms
  • Automated, unattended electronic delivery of mission-critical documents from SAP R/3, mySAP.com and mySAP Business Suite applications
  • Message tracking, notification and delivery status within SAP applications
  • Extensibility from the desktop to the back office - all from one server
  • Wireless access to fax and email
  • PDF file conversion
  • World-class Captaris sales and technical support.
