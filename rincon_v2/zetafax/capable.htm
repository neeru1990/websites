Zetafax

Capabilities

Sending faxes

Send faxes in a variety of ways to suit your needs and those of your organization. Zetafax offers the widest choice of client applications that can be deployed in a single network environment for sending and receiving faxes.

From the Zetafax client
If Zetafax is used on a standalone basis, faxes can be sent directly from the Zetafax client. Modeled on Microsoft Windows XP, the client is attractive, functional and highly intuitive, keeping training and internal support requests to a minimum.

From email
All Zetafax systems include an email gateway that supports Microsoft Exchange 5.5 and 2000, Lotus Domino and SMTP email systems. The email gateway enables email users to send and receive faxes just like email.

From the web client
The Zetafax web client allows network administrators to deploy Zetafax to all network users without the need to install a separate fax client, thus speeding up deployment to multiple users and reducing administration costs.

Faxing your Microsoft Office ® documents directly
Zetafax is integrated with Microsoft Windows XP Professional and Microsoft Office XP, so users can fax direct from Word, Excel or PowerPoint.

Fax directly from any departmental application
Zetafax can be used more widely by fax-enabling other applications that your staff uses with the Zetafax API Toolkit. For example, by integrating Zetafax with your accounts package, your finance staff can fax purchase orders and invoices straight from their PCs.

Receiving faxes

Zetafax can deliver inbound faxes in a variety of ways to suit differing requirements. DID routing comes as standard with all Zetafax starter systems. It requires suitable hardware and telephone lines.

To individual PCs
For more private and immediate delivery, faxes can be routed directly to individual PCs, or to specific departments using the Zetafax DID routing feature. Each user can be allocated a DID 'virtual fax' number which does not require its own dedicated phone line.

To a designated individual
A simple method is to assign one person at your company to receive all faxes. This person can then forward faxes via email to intended recipients.

To a department
By assigning a DID number to a department, rather than an individual, faxes can be sent to specific groups of users. Using the automatic printing feature, faxes can be printed to a nearby printer, for an ideal way of replacing departmental fax machines.

To a central printer
Faxes can be delivered to any network printer and printed automatically.

Administrator Features

Zetafax starts paying for itself from the minute it is installed, and will save you more time and money as you add more users. The return on your investment in Zetafax will easily justify your initial outlay.

Web client for fast deployment to multiple network users
The web client allows network administrators to deploy Zetafax without the need for a separate fax client application install. It is deployed with a single central install by hosting the client application centrally.

Support for clustered Microsoft Exchange environments
The Zetafax connector can be installed on multiple nodes in a cluster server environment. If the Microsoft Exchange server fails over to another node, the Zetafax connector supports the fail over, ensuring continuous availability. Standard with Zetafax Corporate Edition only.

Fax queue management
The Zetafax client allows administrators to abort or rush items in the fax queue belonging to other users. This feature helps customers doing high volume faxing or fax broadcasting.

Automatic printing of failed faxes
The Zetafax server can print faxes that have failed automatically so the printed message can be sent by an alternative method such as regular mail.

Remote administration
The Zetafax Server Monitor and configuration programs can be managed remotely from the Administrator desktop. Most Zetafax server functions can be performed remotely and Administrators can enable or disable functions for remote use.

Report writing and event logging
The Zetafax report writer creates graphical and tabular graphs to enable easy tracking of fax usage and charges to customers and clients. Charge codes can be logged against specific individual, client or departmental charge codes to ensure accurate tracking and allocation of fees, disbursements and other costs.

User Features

Zetafax is about easy, intuitive and quick faxing from the smart, Microsoft Windows XP-style client. With a multitude of methods for sending and receiving faxes, Zetafax is featured to make faxing as straightforward as email. And with the SMS add-on Zetafax also brings text messaging to the desktop.

Intuitive client user interface
The Zetafax client user interface icons, menus and toolbars align Zetafax with the Microsoft Windows/Office XP look and feel, making it easy and intuitive to use.

Multiple ways to automate faxing
Zetafax automates faxing from Microsoft Office documents, GoldMine Business Contact Manager 6.0 and legal case management applications as well as from Microsoft Exchange/Outlook, Lotus Domino/Notes and SMTP email systems. Using the API Toolkit, virtually any other application can be fax-enabled to automate faxing from the desktop.

Easy addressing of faxes
Zetafax uses the Microsoft Outlook Contacts address book to address faxes. Numbers can also be stored public and private address books.

Multi-document faxing
The multi-doc feature allows users to combine multiple documents such as letters, spreadsheets and presentations into a single fax.

Broadcast faxing
The enhanced embedded addressing features of Zetafax allows users to perform large volume fax shots quickly and easily using mail merge.

Multilingual capabilities for international business
The Zetafax client supports the following languages: Danish, Dutch, English (UK), English (US), French, German, Greek, Hebrew, Italian, Norwegian, Polish, Portuguese & Spanish.

Easy to use help features
The Zetafax client help files feature an intuitive menu structure so users can make the most of the rich features of the Zetafax client.
