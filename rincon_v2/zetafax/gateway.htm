Zetafax

E-Mail Gateway

Zetafax allows users to send and receive faxes using their email client. The email gateway supports Microsoft Exchange 5.5 and 2000, Lotus Notes/Domino and SMTP email systems.

  • Sending faxes via the email gateway.
    Faxes can be written and sent from within your chosen email client. The required letterhead and coversheet can be selected as normal and the fax previewed prior to sending. Users are notified by email of successful transmission.

  • Sending attachments via the email gateway
    Using their email client, Zetafax users can send attachments e.g. Microsoft Word or Adobe Acrobat files with faxes just as they do with email. Zetafax converts attachments to TIFF format for faxing.

  • Receiving faxes via the email gateway
    Using the Zetafax direct dial inward (DID) routing option, faxes can be routed straight to the email Inboxes of individual users. If preferred, faxes can be routed to departments or designated administrators.
Using Zetafax in an application server environment

The Zetafax client can be deployed to users with software running on application servers including Microsoft Windows 2000 Terminal Services and Windows Server 2003 Terminal Services, Windows NT 4.0 Terminal Server Edition and Citrix MetaFrame XP.

  • Simpler deployment
    When used in an application server environment, Zetafax is easier and simpler to deploy as it is not necessary to install it on every client device. Network faxing capabilities can be provided to LAN and remote users and to branch offices via a single copy of the Zetafax client on the application server.

  • Faster upgrades
    By upgrading the Zetafax client software on the application server, administrators can ensure that each Zetafax user automatically runs the latest version, eliminating the need to upgrade every desktop.

  • Easy account configuration
    New user accounts can easily be set up on the network. Users have their own personalized settings for the Zetafax client as well as private in-boxes and out-boxes, which can be protected by using file server permissions.

  • Cross platform support
    Using MetaFrame and the Citrix ICA client, the Zetafax client can be used in an application server session on a wide range of operating systems including Macintosh, UNIX, Handheld PCs and Network Terminals.

  • Support for mixed desktop environments
    The Zetafax client can work seamlessly in a mixed desktop environment. By installing the Zetafax client on the local computer, users can submit faxes from an application on an application server client, as well as from an application on the local PC.

  • Easy use anywhere
    Network users can be provided with faxing capabilities in branch offices or at remote sites and can utilize Zetafax using existing thin client sessions over the WAN or using remote access. For fax broadcasting or other large volume faxing, Zetafax supports embedded addressing in the application server environment, meaning easier, faster fax addressing.
