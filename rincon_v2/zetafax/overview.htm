Zetafax

Overview

Zetafax is dependable, easy to use, scalable and feature rich. It is used by over 50,000 organizations in 105 countries, and provides desktop faxing capability, which can be tailored to your needs with a rich range of add-on features. It can also be embedded in 3rd party applications by customers for automated faxing from existing software and by Independent Software Vendors to add value to their own software.

Zetafax for Networks provides faxing from the desktop for networked employees, starting with just 5 users and scaling to an unlimited number of seats. It is very easy to install, configure and manage and highly intuitive for users. Zetafax for Networks is available in three Editions for different sizes of deployment:

  • Business Edition (5-49 users)
  • Corporate Edition (50-unlimited users)
  • Enterprise Edition (minimum 100-unlimited users)
Business Edition

The Zetafax Business Edition is the Equisys entry level network fax software for 5-49 network users.  It is an all in the box software solution, can be installed on existing hardware and is very easy to manage once setup.

The Business Edition includes:

  • A single fax server
  • Zetafax Client for 5, 10 or 25 users
  • Support for 2 fax lines
  • Email Gateway
  • Web Client

The intuitive Microsoft Outlook 2003-style Zetafax Client allows users to write, address and send faxes with little or no training.

Corporate Edition

Zetafax for Networks - Corporate Edition provides fast, easy faxing for 50 or more network users. It provides an all in the box solution, and features an easy to use Zetafax Server installation and configuration wizard and the ability to deploy multiple clients across the business using a single, central install.

The Corporate Edition includes:

  • A single Zetafax Server
  • Zetafax Clients for 50, 100 or an unlimited number of users
  • Support for two fax lines
  • Zetafax Web Client
  • Zetafax Email Gateway
  • DDI routing
  • Least Cost Routing
  • Support for Exchange clusters and Windows child domains for the Zetafax Email Gateway
  • Software Assurance for one year
  • Queue management
  • Remote monitoring
  • Report writing
  • Active Directory integration
Enterprise Edition

Zetafax for Networks - Enterprise Edition provides fast, easy faxing for a minimum of 100 users. For greatest flexibility, Zetafax Enterprise licenses support:

  • Any number of sites
  • Unlimited servers
  • Unlimited lines
  • Zetafax Client for unlimited users
  • Zetafax Email Gateway
  • Zetafax Web Client

The Zetafax Server installation used MSI Install and an installation and configuration wizard. Support for Group Policy means that if desired, Zetafax Clients can be installed enterprise-wide with a single, centralized install from one workstation. Additional features of the Enterprise Edition include:

  • Zetafax SMS Gateway
  • Zetafax API Toolkit
  • Zetafax Document Conversion add-on for API, Email Gateway and Web Client
  • Software Assurance for one year
